# Yuzuki
PoolC 홈페이지 제작 프로젝트

## Requirements
- python 2.x (Recommend 32bit on windows)

## Development Guide
1. install dependencies  
  `pip install -r requirements.txt`
2. init database  
  `python yzk_setup_poolc.py`  
  Default admin account info : `admin / asdf1234`
3. run server  
  `python main.py`  
  now Yuzuki is serving on 8080 port.
